动态数据绑定:如何监听其属性的读取与变化？
也就是说，如何知道程序访问了对象的哪个属性，又改变了哪个属性？

问题

下面的例子如何实现?
let app1 = new Vue({
  name: 'youngwind',
  age: 25
});

let app2 = new Vue({
  university: 'bupt',
  major: 'computer'
});
要实现的结果如下：
app1.data.name 
// alert你访问了 name
app1.data.age = 100;  
// alert你设置了 age，新的值为100
app2.data.university 
// alert你访问了 university
app2.data.major = 'science'  
// alert你设置了 major，新的值为 science
