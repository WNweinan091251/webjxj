##### 编写一个javscript函数 fn，该函数有一个参数 n（数字类型），其返回值是一个数组，该数组内是 n 个随机且不重复的整数，且整数取值范围是 [2, 32]。

## 境界层次:
1. 可用
1. 健壮
1. 可靠
1. 宽容
1. 求精

```
function fn(n) {
	var an = testN(n);
	if (!an) {
		console.log('fn n error');
		return
	}
	var arr = [];
	for (var i = 0; i < n; i++) {
		arr[i] = Math.floor(Math.random() * 31 + 2);
	}
	return arr;
}

function testN(n) {
	if (typeof n == 'number') {
		if (n % 1 == 0) {
			return 1;
		} else {
			return 0;
		}
	} else if (typeof n == 'string') {
		if (/^\d*$/.test(n)) {
			return 1;
		} else {
			return 0;
		}
	} else {
		return 0;
	}
}
```
